# Remote Access SOP (Demo)

**Scope:** VPN + bastion access to GCP.  

**Steps:**

1. Request & approve remote access in ticketing.  
2. Enforce MFA on VPN & IdP.  
3. Monitor login events in SIEM; alert on non-MFA or geo-anomaly logins.  

**Mapped controls:**  

- CMMC: AC.L2-3.1.12  
- NIST: AC-17, IA-2  
- SOC 2: CC6.2  
- ISO: A.5.17
